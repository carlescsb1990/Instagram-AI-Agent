## 🚀 COMANDOS DE OPERACIÓN

### Desarrollo

```bash
# Iniciar servidor de desarrollo
npm start

# Ver logs en tiempo real
npm run logs

# Verificar salud del sistema
npm run health

# Ejecutar tests
npm test
```

### Producción

```bash
# Iniciar con PM2
npm run pm2:start

# Ver status
pm2 status riona-ai-agent

# Ver logs
pm2 logs riona-ai-agent

# Restart
npm run pm2:restart
```

### Docker

```bash
# Build imagen
npm run docker:build

# Ejecutar contenedor
npm run docker:run
```
